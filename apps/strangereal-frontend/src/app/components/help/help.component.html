Navigating the Map

Click and drag to pan around and scroll to zoom. Hover over any markers to view what kind of marker it is and the name of the unit if it was provided.

Creating Units

Click anywhere on the map to bring up the new unit dialog box. Once the unit details are filled out and submitted, a marker will be placed at the location that was clicked.

Moving Units

Units can be moved by holding Shift, then dragging the marker to the desired new location.

Removing Units

Units can be removed from the map by holding Ctrl, then clicking the undesirable unit. A confirmation dialog box will appear to confirm the removal.

Keyboard Shortcuts

Numerous keyboard shortcuts are available to speed up creation of units. Press Shift + / or select the "Keyboard Shortcuts" option from the context menu.

Syncing Changes to the Server

All changes to the map are automatically synced to the server. Markers will appear slightly transparent and desaturated to indicate that they are still syncing with the server. They will return to the normal color once the new unit or change to an existing unit has been synced.

You must refresh the page to retrieve changes made on other devices or from other users. Failure to do so could cause changes to be overridden or lost. Live collaboration is a planned feature for the future.
